# Quick Setup Script for Firebase Cloud Functions

## 🚀 Run These Commands:

```bash
# 1. Install Firebase CLI (if not installed)
npm install -g firebase-tools

# 2. Login to Firebase
firebase login

# 3. Initialize Functions in your project
cd c:\Users\asus\AI_Projects\Innovations
firebase init functions

# Select:
# - Use existing project: nextgen-innohub
# - Language: JavaScript
# - ESLint: No
# - Install dependencies: Yes

# 4. Replace functions/index.js with the code from functions-index.js
# Copy content from: functions-index.js
# Paste into: functions/index.js

# 5. Deploy functions
firebase deploy --only functions

# 6. Done! Your functions are live!
```
